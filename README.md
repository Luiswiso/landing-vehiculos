# landing-vehiculos

Landing page para servicio de mantenimiento de flota pesada.

**Stack:** Astro 5 · Tailwind CSS 4 · React · Node.js SSR · Playwright E2E

**Demo:** https://landing-vehiculos.vercel.app

## Comandos

| Comando           | Acción                              |
| :---------------- | :---------------------------------- |
| `npm install`     | Instala dependencias                |
| `npm run dev`     | Dev server en `localhost:4321`      |
| `npm run build`   | Build de producción en `./dist/`    |
| `npm run preview` | Preview del build local             |
| `npx playwright test` | Ejecuta suite E2E completa      |
